# webExe
## GruntItem
* 安装grunt-cli<br>
打开控制台（`windows下使用管理员权限打开`）输入`npm install -g grunt-cli`<br>
* Package.json<br>
写下网站或系统的名称和版本
```
    {
      "name":"webExe",
      "version":"1.0.0",
      "devDependencies":{}
    }
```
* 安装grunt<br>
在`D：\webExe`目录下输入`npm install grunt --save-dev`<br>
`“—save-dev”的意思是，在当前目录安装grunt的同时，顺便把grunt保存为这个目录的开发依赖项`
